\asubsection{1}{Introduction to fundamental data types}

C++ 对于很多不同的数据类型具有内置的支持。
这些被称为\textbf{基本数据类型} fundamental data types，
不过也经常被不正式的称为 \textbf{basic types}，\textbf{primitive types}，
或者 \textbf{built-in types}。
