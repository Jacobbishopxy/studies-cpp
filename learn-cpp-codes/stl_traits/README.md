# STL traits

![STL](./stl.png)

图片出自：[CPlusPlusThings](https://github.com/Light-City/CPlusPlusThings/blob/master/src_analysis/stl/traits.md)

- [iterator_tags](https://en.cppreference.com/w/cpp/iterator/iterator_tags)：仅作为标记（结构体继承的方式）标记不同类别的容器
