\asubsection{2}{The debugging process}

一旦问题被识别，调试问题通常由五个步骤组成：

\begin{enumerate}
  \item 寻找问题的根本原因（通常是不能运行的那一行代码）
  \item 确保理解为什么该状况会发生
  \item 决定如何修复该状况
  \item 修复导致该状况的问题
  \item 重新测试确保问题已被解决，并且没有新的问题产生
\end{enumerate}
